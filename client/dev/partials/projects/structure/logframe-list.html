Un cadre logique est un document qui décrit les objectifs d'un projet, les activités misent en oeuvre pour y parvenir, ainsi que les indicateurs qui permette de suivre l'avancement de chaque élément

Tous les indicateurs présents dans les cadres logiques doivent être calculables à partir des données décrites dans les sources de données
