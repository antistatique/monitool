Si tous les membres de Médecins du Monde peuvent consulter les données de tous les projets, il n'en est pas de même pour les droits de modifications

Pour chaque projet, deux groupes d'utilisateurs existent:

  • Les propriétaires: Peuvent modifier le cadre logique, les lieux et groupes d’activité, les formulaires de collecte.
  • Les opérateurs de saisies : Peuvent saisir des données sur le projet.
